CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • Everything on a webpage is essentialy a box/block

  • In CSS we broadly have two types of boxes — block boxes and inline boxes
  • You can set various values for the display type using the display property
  • Block boxes break onto a new line
  • Width and height are not affected by padding margins and borders
  • Instead they push other elements away from the box
  • The box will usually share the width of its container by default
  • Contrastingly Inline boxes do not break onto a new line
  • Padding, margins, and borders apply but dont push away other inline boxes
  • There are two CSS box models — Standard and Alternative
  • In the standard CSS box model the total size of the box is affected by paddings and borders
  • In the alternative model the width is literal and the padding and border arent calculated seperately for a total
  • The margin is not counted towards the actual size of a box

Git

  • ls - Lists directories
  • pwd - Checks current directory
  • cd - Changes directory
  • Touch - Creates a file (name.extension)
  • Start - Opens a file (name.extension)
  • ls -al ~/.ssh - Checks for existing SSH keys
  • ssh -keygen -t ed25519 -C "githubemail.com" - Generates SSH key
  • git status - Checks what branch you are currently on
  • git checkout "branchname" - Moves working branch to named branch
  • git clone "SSH value" (inside directory) - to create a clone
  • git checkout -b "branchname" - Creates a new branch and moves working branch there
  • -a - All
  • git pull origin main - Receive branch modifications to local environment
  • - Origin indicates source (githubrepo)
  • - main is the branchname
